Usage tips
  1. Load the Instagram HTML export file (file → open), or paste the HTML into the dropzone.

  3. Click Copy Markdown to get a clipboard-ready list for pasting into your other app.
  4. If you need automatic screenshots, use a headless tool (Puppeteer) or a screenshot browser extension and drag the file in here.
